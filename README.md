# TinyMaker-Open-Source-3D-Printer
:speech_balloon:This repository is not yet complete. At the moment, we have only uploaded part of the available materials, including firmware, libraries, schematics, and 3D models.

Progress is moving a bit slowly for two main reasons: first, we are currently fully occupied with fulfilling the second batch shipments on Indiegogo; second, we tend to be quite detail-oriented and perfectionist, so we want to make sure everything is properly organized, clearly structured, and easy to understand before releasing it.

We appreciate your patience and support, and we will continue improving the repository as quickly as possible.

If you plan to use the firmware provided here, please make sure to use the correct version of the libraries (the versions we uploaded are the verified ones), and also use Arduino IDE version 1.8.19.
